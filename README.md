# JWM Setup: ชุดคิตกู้ชีพคอมพิวเตอร์สเปกต่ำ (Thai Edition) 🚀

บันทึกช่วยจำสำหรับตัวเองเวลาลง Linux เครื่องใหม่ (เพราะฮาร์ดดิสก์ลูกเก่าเริ่มเต็ม) หรือเอาไว้แชร์ให้คนไทยที่อยากใช้ Window Manager ตัวเล็กแต่พร้อมซิ่งทันที

### 📦 ฟังก์ชันเด่น (เซ็ตระบบไว้ให้ครบแล้ว)
1. พิมพ์ไทยได้ทันที สลับภาษาด้วย `Left Alt + Shift`
2. ระบบสลับหน้าต่างคู่ขนาน: `SUPER + Tab` (ผ่านโปรแกรม alttab) หรือ `ALT + Tab` (ระบบภายใน jwm)
3. สตาร์ทเมนูอัปเดตอัตโนมัติทุกครั้งที่สั่ง รีสตาร์ท JWM (พึ่งพา xdgmenumaker)
4. หน้าต่าง Terminal มอนิเตอร์ความร้อน CPU/GPU, ปริมาณ RAM, และพื้นที่ HDD คงเหลือแบบเรียลไทม์

---

### ❌ สรุปสิ่งที่ลองแล้ว "ไม่เวิร์ค" (ห้ามหาทำซ้ำเด็ดขาด!)
* **อย่าดันทุรังเซ็ต alttab ผูกกับ Alt+Tab ตรงๆ ใน JWM Config:** ไม่เวิร์ค! ระบบ X11 จะฟ้อง Error เรื่อง mask ทันที เพราะ JWM มันจองปุ่มนี้ไว้ในระดับฮาร์ดโค้ด ให้เลี่ยงไปใช้ปุ่มหลบอย่าง `SUPER+Tab` ผ่านสคริปต์สตาร์ทอัพแทน นิ่งกว่าเยอะ
* **อย่าสั่ง killall xdg-desktop-portal-gtk เพื่อหวังปลดล็อกปุ่ม:** ไม่เวิร์ค! เพราะ Linux Mint มี systemd คอยเฝ้าสั่งชุบชีวิตมันกลับมาในวิเดียว และมันไม่เกี่ยวอะไรกับคีย์บอร์ดแต่แรกด้วย
* **ห้ามพิมพ์คำสั่งระบบใน JWM Config ด้วยอักษรตัวใหญ่ (เช่น Next/Prev):** ไม่เวิร์ค! ตัวสแกน Syntax ของ JWM บังคับใช้ตัวพิมพ์เล็กล้วนเท่านั้น (`next` และ `prev`) ไม่งั้นพ่น Warning พังใส่หน้าทันที
* **คำสั่ง alttab ห้ามระบุอาร์กิวเมนต์ `-t`:** เออร์เรอร์ทันที โปรแกรมไม่มีออปชันนี้
